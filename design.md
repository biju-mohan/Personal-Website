## Typography Hierarchy

```
H1 (Name):        48px / 3rem     - Bold
H2 (Sections):    32px / 2rem     - Semi-bold
H3 (Job titles):  24px / 1.5rem   - Semi-bold
Body text:        16px / 1rem     - Regular
Small text:       14px / 0.875rem - Regular
```

## Interactive Elements

### Navigation (Desktop)
```
┌─────────────────────────────────────────────────────┐
│ [About] [Experience] [Education] [Contact]          │ ← Hover: underline
└─────────────────────────────────────────────────────┘
```

### Navigation (Mobile - Expanded)
```
┌─────────────────────────┐
│ [Your Name]          ×  │ ← Close button
├─────────────────────────┤
│                         │
│ About                   │ ← Touch-friendly sizing
│                         │
│ Experience              │
│                         │
│ Education               │
│                         │
│ Contact                 │
│                         │
└─────────────────────────┘
```

### Button States
```
Default:  [ Get In Touch ]
Hover:    [ Get In Touch ] ← Slightly darker background
Active:   [ Get In Touch ] ← Pressed state
```

### Form Validation
```
✓ Valid field:   Email: [user@example.com] ← Green border
✗ Invalid field: Email: [invalid-email]    ← Red border + error message
```

## Spacing & Layout Grid

```
Desktop Container: Max 1200px centered
Mobile Container:  100% width with 20px padding

Section spacing:   80px between sections
Element spacing:   24px between elements
Text spacing:      16px between paragraphs
```

## Animation Concepts

1. **Smooth scroll** to sections (500ms ease-in-out)
2. **Fade-in** sections on scroll (subtle)
3. **Button hover** effects (150ms transition)
4. **Form focus** states (200ms transition)
5. **Mobile menu** slide-in (300ms ease)

---

*This design prioritizes simplicity, readability, and professional appearance while being fully responsive and accessible.*
